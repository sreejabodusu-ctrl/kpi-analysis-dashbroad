# 📊 Sales KPI Dashboard

An interactive Sales KPI Dashboard built using Streamlit, Pandas, NumPy, and Plotly. This project provides real-time insights into sales performance through key business metrics, visualizations, and interactive filters.

---

## 🚀 Features

* 📈 Total Sales KPI
* 💰 Total Profit KPI
* 📦 Total Orders KPI
* 🛒 Average Order Value (AOV)
* 📊 Profit Margin Analysis
* 📅 Monthly Sales Trend Visualization
* 🏷️ Category-wise Sales Analysis
* 🌍 Regional Sales Distribution
* ⭐ Top Products Performance
* 🔍 Interactive Filters
* 📥 Download Filtered Dataset

---

## 📋 Dataset

The project uses a sample sales dataset containing the following fields:

| Column     | Description             |
| ---------- | ----------------------- |
| Order_ID   | Unique Order Identifier |
| Order_Date | Date of Order           |
| Product    | Product Name            |
| Category   | Product Category        |
| Region     | Sales Region            |
| Sales      | Sales Amount            |
| Profit     | Profit Earned           |

---

## 📊 KPIs Included

### 1. Total Sales

Displays the total revenue generated.

### 2. Total Profit

Displays the overall profit earned.

### 3. Total Orders

Shows the number of orders placed.

### 4. Average Order Value (AOV)

Average revenue generated per order.

### 5. Profit Margin

Percentage of profit earned relative to sales.

---

## 📈 Dashboard Visualizations

### Monthly Sales Trend

Tracks sales performance over time.

### Sales by Category

Analyzes sales across product categories.

### Regional Sales Distribution

Shows sales contribution by region.

### Top Products Analysis

Highlights the best-performing products.

### Profit by Region

Compares profitability across regions.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/sreejabodusu-ctrl/Sales-KPI-Dashboard.git
cd Sales-KPI-Dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
